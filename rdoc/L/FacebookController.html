    class L::FacebookController

    Kontroler wszystkich akcji facebookowych.

    W aplikacji dziedziczy po nim FacebookController, a po nim powinien dziedziczyć każdy kontroler obsługujący akcje aplikacji facebookowej na płótnie (canvas).

    Zaloguj użytkownika facebookowego do aplikacji, jeśli żaden nie jest zalogowany.

    # File app/controllers/l/facebook_controller.rb, line 17
def sign_in_fb_user
  if request.post? and current_user.blank?
     redirect_to user_omniauth_authorize_path(:facebook, back_to: request.fullpath)
  elsif current_user.blank?
    redirect_to $fb_app_url
  end
end
