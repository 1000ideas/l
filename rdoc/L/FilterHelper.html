    • Zwraca:

      • nazwę pola (lub nazwę pola poprzedzoną nazwą tabeli) i porządek sortowania (DESC/ASC) przekazany przez parametry, lub pusty String w przypadku gdy nie wybrano sortowania

    # File app/helpers/l/filter_helper.rb, line 33
def sort_order(table_name = nil)
  return '' if params[:sort].blank?
  regex = %r{^([a-z_]*)_(asc|desc)$}
  return '' unless params[:sort].match regex
  name = $1
  type = $2.upcase
  table_name.blank? ? "`#{name}` #{type}" : "`#{table_name}`.`#{name}` #{type}"
end
